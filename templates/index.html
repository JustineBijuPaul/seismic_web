About the Project

The Seismic Detection Project utilizes advanced machine learning algorithms to analyze audio and seismic data. By extracting features from the data and training a logistic regression model, we can accurately predict the presence of seismic events. Our model is trained on a diverse dataset of seismic and non-seismic events, ensuring high accuracy and reliability. The project is designed to be user-friendly, allowing users to upload their data and receive predictions in a matter of seconds.

Used Technology

  • Machine Learning
  • Python
  • Flask
  • Odesy

Features

  • Real-time seismic detection from audio and seismic data.
  • User-friendly interface for uploading and analyzing data.
  • Visualization of waveform data with highlighted seismic events.
  • Option to download seismic hit points as MSEED files.
